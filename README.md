# JavaScript Core Fundamentals

## Purpose

created to strengthen my understanding of the language before moving to
DOM manipulation and frameworks.

## Topics Covered
- Variables and data types
- Functions and scope
- Conditionals and loops
- Arrays and objects
- Closures and scope behavior

## Why This Repo Exists
I intentionally paused progression to reinforce foundational concepts
and ensure long-term confidence when building applications.
